Frontend

Possuo base sólida no desenvolvimento de interfaces responsivas, garantindo que as aplicações se adaptem corretamente a diferentes tamanhos de tela, como desktops, tablets e dispositivos móveis.

Desenvolvo com foco em usabilidade e interface intuitiva, aplicando boas práticas de UI/UX para melhorar a navegação, organização visual e experiência do usuário.

Possuo noções básicas de SEO, como uso correto de tags semânticas, hierarquia de títulos e acessibilidade, visando melhor indexação e desempenho nos motores de busca.

Realizo o consumo de APIs utilizando fetch, tratando respostas e exibindo dados dinamicamente na interface de forma clara e eficiente.

Backend

Tenho noções em lógica de programação, utilizando variáveis, operadores, estruturas condicionais, laços de repetição, contadores e algoritmos para resolver problemas de forma eficiente.

Entendo o conceito de arquitetura de aplicações seguindo padrões como Controllers, Services, Repositories, Middlewares e Routes, promovendo organização, manutenibilidade e escalabilidade do código.

Desenvolvo APIs RESTful utilizando comunicação via HTTP, troca de dados em formato JSON, aplicando validação de dados, tratamento de erros e boas práticas de segurança.

Tenho conhecimento em autenticação utilizando JWT, além de noções de performance, escalabilidade, estrutura de dados e paradigmas da Programação Orientada a Objetos (POO).

Utilizo versionamento de código com Git, aplicando conceitos como branches, commits, merge, pull, stage, diff, log e noções básicas de rebase.

Banco de Dados

Tenho conhecimento em modelagem de banco de dados relacional, utilizando conceitos como Primary Key, Foreign Key, UNIQUE, índices e relacionamentos (1:1, 1:N, N:N).

Sei realizar consultas SQL como SELECT, INSERT, CREATE, ALTER, DELETE e DROP, além do uso de JOINs, Views, Stored Procedures, Trigger Functions e controle de transações com rollback.

Tenho atenção à integridade e organização dos dados, aplicando boas práticas de normalização, uso adequado de índices para otimização de consultas e cuidado com consistência das informações.
